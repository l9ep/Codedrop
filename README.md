# CodeDrop

CodeDrop is a code sharing site made by **Yuki**.

The idea is simple — you paste your code, give it a title, and share it with anyone using a link. No sign up needed to drop code, but you can make an account to manage your drops, like other people's code, leave comments, and build a profile.

It was built because most code sharing sites are cluttered, slow, or just ugly. CodeDrop is clean, fast, and dark by default.

---

## What it does

- Paste and share code snippets with a single link
- Supports 24 programming languages
- Public, unlisted, or private drops
- Raw view — just the plain code, nothing else
- Like drops, leave comments, clone someone else's drop
- User profiles with avatars, bios, and stats
- Search and explore public drops from the community
